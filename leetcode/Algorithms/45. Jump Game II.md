## 45. Jump Game II

```c++
class Solution {
public:
    int jump(vector<int>& nums) {
        int position = nums.size() - 1;
        int steps = 0;
        
        while(position != 0) {
            for(int i = 0; i < position; i++) {
                if(nums[i] >= position - i) {
                    position = i;
                    steps++;
                    break;
                }
            }
        }
        
        return steps;
    }
};
```

- 점프 횟수의 최소를 찾기 위해서는 `position(마지막위치로 초기화)에 도달할 수 있는 가장 왼쪽 노드를 찾는다.` 가 기본 아이디어이다.
