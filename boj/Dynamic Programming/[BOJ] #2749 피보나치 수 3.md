## [BOJ] #2749 피보나치 수 3

[문제](https://www.acmicpc.net/problem/2749)



| 메모리   | 시간  |
| -------- | ----- |
| 13736 KB | 12 ms |

```c++
#include <iostream>
using namespace std;
long long d[1500000];
int main() {
	d[0] = 0;
	d[1] = 1;
	for (int i = 2; i < 1500000; i++) {
		d[i] = d[i - 1] + d[i - 2];
		d[i] %= 1000000;
	}

	long long n;
	cin >> n;
	cout << d[n % 1500000] << '\n';

	return 0;
}
```

- 피보나치 수를 M으로 나눈 나머지는 주기를 가지고 이를 피사노 주기라고 한다.

- 주기의 길이가 K이면

- N번째 피보나치 수를 M으로 나눈 나머지는 N%K 번째 피보나치 수와 같다.

- $$
  M = 10^k
  $$

   일 때, k > 2라면, 주기는 항상 
  $$
  15*10^{k-1}
  $$
  이다.
